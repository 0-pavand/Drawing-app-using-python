🎨 My Paint Game (Mini Project 2)

This is a simple Paint Application built using Python and Pygame.
It allows users to draw freely on the canvas with customizable brush sizes and colors.
The project is created as part of my second mini project to learn event handling, graphics rendering, and UI design in Python.

🚀 Features

🖌️ Multiple Brush Sizes (Extra Large, Large, Medium, Small)

🎨 Different Colors for drawing

📜 Drawing History saved in memory until the program closes

🖼️ Interactive Menu Bar for easy brush and color selection

⚡ Smooth Drawing using mouse events

🛠️ Technologies Used

Python 3.x

Pygame Library

📸 Preview

When you run the project, you’ll see a canvas with a menu bar where you can:

Select brush size (left side)

Select colors (menu bar buttons)

Draw anywhere on the canvas using your mouse
